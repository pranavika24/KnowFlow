# 🚀 KnowFlow - AI Knowledge Base

## 📌 Overview
KnowFlow is an AI-powered knowledge base application that allows users to upload PDF documents and ask questions about them. It uses Retrieval-Augmented Generation (RAG) to provide accurate answers based on document content.

---

## 🎯 Features
- 📄 Upload and process multiple PDFs
- 🔍 Semantic search using FAISS
- 🤖 AI-powered question answering
- 💬 Chat-style interface
- 🧠 Context-aware responses
- 💾 Local embedding storage

---

## 🛠️ Tech Stack
- Python
- Streamlit
- HuggingFace Transformers
- FAISS (Vector Database)
- PyPDF

---

## ⚙️ How It Works
1. Upload PDF documents
2. Extract text from files
3. Split into chunks
4. Convert into embeddings
5. Store in FAISS vector database
6. Retrieve relevant chunks based on query
7. Generate answer using AI model
